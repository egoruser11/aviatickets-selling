# Система продажи авиабилетов

Курсовое Laravel-приложение для работы с PostgreSQL. Реализованы связанные таблицы `airlines`, `flights`, `tickets`, начальные данные и формы по требованиям лабораторной работы.

## Что есть в приложении

- Форма 1: главное меню.
- Форма 2: авиакомпании, поиск, добавление, удаление, изменение и связанные рейсы.
- Форма 3: рейсы, добавление, удаление и изменение.
- Форма 4: параметризованные запросы по авиакомпаниям, рейсам и билетам.
- Форма 5: билеты, выпадающий список рейсов вместо ручного ввода внешнего ключа, сортировка и прокрутка таблицы.

## Запуск

```bash
docker compose up -d --build
docker compose exec app composer install
docker compose exec app php artisan key:generate
docker compose exec app php artisan migrate:fresh --seed
```

После запуска приложение доступно по адресу:

```text
http://localhost:8877
```

PostgreSQL проброшен на локальный порт `5433`.
